# 更新日志

本项目版本遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-06-06

首个公开版本。

### 内容
- 全书 **18 章**（教材级）+ 附录 A（环境配置）/ B（数学回顾）/ C（数据字典）/ D（术语与符号表）：
  - 第一部分·基础：导论、Python 工具栈、数据获取、探索性分析
  - 第二部分·金融计量：收益与风险、时间序列、因子模型、面板回归
  - 第三部分·机器学习：监督学习、特征工程、树模型、深度学习、NLP
  - 第四部分·应用与前沿：组合优化、回测、信用风险、大模型、金融智能体
- **19 个配套 Jupyter Notebook**，离线可运行、习题附参考解答；
- **5 个内置数据集**（股票/市场/因子/财务面板/信用）；`fds` 复用工具包；
- **25 张正文配图**（统一主题，`scripts/make_figures.py` 可复现）。

### 工程
- 环境：uv 管理，推荐 Python 3.14（兼容 3.11+），`uv.lock` 锁定版本；
- 代码：PEP8 + 类型注解（ruff）、冒烟测试（pytest）；
- 第15章引入 akquant 事件驱动回测；第17-18章可接真实 LLM（openai 兼容）；
- 成书：MkDocs + Material，GitHub Actions 自动部署到 GitHub Pages；
- 在线运行：每章 Colab / Binder 入口；
- 可导出整书 PDF（`scripts/build_pdf.py`）。
