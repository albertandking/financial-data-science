# 第13章 金融文本与 NLP

!!! info "配套代码"
    `notebooks/ch13_nlp_text.ipynb`（使用 jieba / transformers，需 `--extra advanced`）

## 13.1 学习目标

- 中文金融文本处理（分词、清洗）
- 情感分析与舆情指标
- 从年报/新闻中提取信号

## 13.2 内容大纲

1. 中文分词（jieba）与停用词
2. 词袋、TF-IDF 与词向量
3. 金融情感词典与情感打分
4. 预训练模型（中文 BERT/FinBERT）做情感分类
5. 把文本信号并入预测模型

## 13.3 练习

1. 对一批财经新闻标题做情感打分，与次日收益相关性分析。
2. 比较词典法与预训练模型在情感分类上的差异。
