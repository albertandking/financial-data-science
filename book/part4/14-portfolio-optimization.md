# 第14章 投资组合优化

!!! info "配套代码"
    `notebooks/ch14_portfolio_optimization.ipynb`（使用 cvxpy / scipy）

## 14.1 学习目标

- 马科维茨均值-方差优化
- 有效前沿与切点组合
- 约束与稳健优化

## 14.2 内容大纲

1. 组合收益与风险：$\mu_p = w^\top\mu$，$\sigma_p^2 = w^\top\Sigma w$
2. 均值-方差优化与有效前沿
3. 最小方差组合、最大夏普组合
4. 约束：禁止做空、权重上限、行业中性
5. 协方差估计的不稳定与收缩估计

## 14.3 练习

1. 用内置四只股票绘制有效前沿，标出最大夏普组合。
2. 加入禁止做空约束，比较前沿变化。
