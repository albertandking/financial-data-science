# 第17章 大模型与金融前沿

!!! info "配套代码"
    `notebooks/ch17_llm_finance.ipynb`（调用大模型 API，需自备 key）

## 17.1 学习目标

- 了解大语言模型（LLM）在金融中的应用场景
- 用 LLM 做信息抽取、摘要与情感分析
- 认识局限：幻觉、时效、合规

## 17.2 内容大纲

1. LLM 概览：从 Transformer 到指令模型
2. 金融应用：年报问答、研报摘要、事件抽取、舆情
3. 检索增强生成（RAG）接入金融知识库
4. 提示工程与结构化输出
5. 风险与合规：幻觉、数据泄露、可解释性、监管要求

!!! warning "合规与风险"
    LLM 输出可能不准确，**不能直接用于投资决策**；注意数据合规与隐私。

## 17.3 练习

1. 用 LLM 从一段研报中抽取结构化要点（评级、目标价、风险）。
2. 讨论：把 LLM 情感信号并入量化策略需要注意什么？
