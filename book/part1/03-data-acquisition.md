# 第3章 金融数据获取与清洗

!!! info "配套代码"
    `notebooks/ch03_data_acquisition.ipynb`（含「联网」小节，使用 akshare）

## 3.1 学习目标

- 了解中国市场常见数据源（行情、财报、宏观）
- 用 akshare / tushare 抓取 A 股数据
- 处理缺失值、复权、停牌、异常值

## 3.2 内容大纲

1. 数据源概览：akshare（免注册）、tushare（需积分）、Wind/聚宽（机构）
2. 抓取 A 股日线：前复权 vs 后复权
3. 数据清洗：停牌、缺失、涨跌停、异常点
4. 对齐多只标的的交易日历
5. 存储为 Parquet 入库

!!! tip "离线优先"
    本章正文示例默认用内置数据；联网抓取部分单独标注，需 `uv sync --extra data`。
