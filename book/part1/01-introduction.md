## 1.2 什么是金融数据科学

金融数据科学是**用数据驱动的方法研究金融问题**的交叉学科，融合了：

- **金融学**：资产定价、风险管理、投资组合理论
- **统计与计量经济学**：回归、时间序列、假设检验
- **机器学习**：预测、分类、表示学习
- **计算机科学**：数据工程、可视化、可复现研究

与一般数据科学相比，金融数据有几个鲜明特点：

1. **低信噪比**：价格变动中可预测的成分很小，容易过拟合。
2. **非平稳**：市场结构随政策、情绪变化，历史规律可能失效。
3. **厚尾与极端事件**：收益率分布尾部远比正态分布厚。
4. **强烈的时序依赖**：不能随意打乱样本顺序做交叉验证。

!!! warning "数据泄露提醒"
    金融建模最常见的错误是**前视偏差（look-ahead bias）**——在预测时用到了
    未来才能知道的信息。本书会反复强调如何避免它。

## 1.3 第一段代码

下面用内置示例数据集计算四只虚构股票的累计收益并作图。完整可运行代码见配套
notebook，这里展示核心片段：

```python
from fds import load_sample_prices, daily_returns, set_chinese_font
import matplotlib.pyplot as plt

set_chinese_font()                       # 配置中文字体
prices = load_sample_prices()            # 加载内置示例数据
rets = daily_returns(prices)             # 计算日度收益率

cumulative = (1 + rets).cumprod()        # 累计净值
cumulative.plot(title="四只示例股票累计净值")
plt.ylabel("净值（初始=1）")
plt.show()
```
