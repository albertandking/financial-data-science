## 4.2 描述性统计

拿到收益率序列，先看四个矩：

| 统计量 | 含义 | 金融解读 |
|---|---|---|
| 均值 | 平均收益 | 通常很小，接近 0 |
| 标准差 | 波动率 | 风险的主要度量 |
| 偏度（skew） | 不对称性 | 负偏常见（暴跌比暴涨剧烈） |
| 峰度（kurtosis） | 尾部厚度 | 远大于正态的 3（厚尾） |

```python
from fds import load_sample_prices, daily_returns
rets = daily_returns(load_sample_prices())
rets.describe()
rets.skew(), rets.kurtosis()    # kurtosis 为超额峰度，正态=0
```

## 4.3 金融数据的"风格化事实"

实证金融发现收益率序列普遍具有以下特征（stylized facts），它们决定了我们该用什么模型。

### 4.3.1 厚尾（fat tails）

极端涨跌的发生频率远高于正态分布的预测。**QQ 图**是最直观的诊断工具：把样本分位数
与正态分位数对比，若尾部点偏离对角线向外翘起，即为厚尾。

> 实务含义：用正态分布算 VaR 会**低估**极端风险。

### 4.3.2 波动率聚集（volatility clustering）

"大波动跟着大波动，小波动跟着小波动"。收益率本身几乎不自相关，但**收益率的绝对值
或平方**有显著且缓慢衰减的自相关。这正是 GARCH 模型（第6章）要刻画的现象。

### 4.3.3 收益近似不相关

日收益率的自相关通常很弱——这与"市场难以预测"一致。但这**不等于**独立：上面的波动率
聚集说明高阶相依依然存在。

## 4.4 相关性与协动

多资产分析关注它们如何一起涨跌。**相关系数矩阵 + 热力图**是标准工具：

```python
corr = rets.corr()
import seaborn as sns
sns.heatmap(corr, annot=True, cmap="coolwarm", vmin=-1, vmax=1)
```

!!! tip "相关性是时变的"
    危机时资产相关性往往骤升（"同跌"），分散化失效。可用**滚动相关系数**观察其随时间变化。

## 4.5 绘图规范

金融图表要点：

1. **中文不乱码**：统一调用 `from fds import set_chinese_font; set_chinese_font()`。
2. **标注清晰**：标题、轴标签、图例、单位齐全。
3. **配色克制**：避免高饱和大红大绿误导；涨跌色注意中国习惯（红涨绿跌）与国际相反。
4. **可比性**：多图共享坐标轴范围，便于横向比较。

## 4.6 本章小结

- 先看四个矩：均值≈0、标准差是风险、负偏、高峰度（厚尾）
- 三大风格化事实：厚尾、波动率聚集、收益近似不相关
- 多资产用相关矩阵 + 热力图，注意相关性时变
- 中文图表统一字体与规范标注
