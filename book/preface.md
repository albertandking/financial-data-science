# 前言

## 为什么写这本书

金融与数据科学的交叉，正在重塑投资研究、风险管理与金融科技的工作方式。但市面上的资料常常两头不靠：
偏金融的讲不清代码与数据，偏机器学习的又脱离金融的特殊性——低信噪比、非平稳、厚尾、强烈的时序依赖，
以及前视偏差、数据窥探这些「一不小心就把回测做漂亮、实盘亏到怀疑人生」的陷阱。

本书试图把这两条线拧成一股：**用可运行的代码、贴近中国市场的数据，把金融数据科学从概念讲到落地**，
同时反复强调那些「看起来对、其实错」的方法论坑。

## 读者对象与前置知识

面向**经济、金融、统计、计算机等专业的本科高年级学生与研究生**，也适合入门量化的从业者自学。
假设读者具备：

- 初步的 Python 基础（变量、函数、循环）；不熟悉的部分见[第2章](part1/02-python-toolkit.md)；
- 概率统计与微积分的基本概念；需要唤醒记忆见[附录B](appendix/b-math-review.md)；
- 对金融市场的基本认识（股票、债券、收益率）。

满是英文术语？见[附录D 术语与符号表](appendix/d-glossary.md)随时查阅。

## 章节地图

全书四部分层层递进，附录可随时回查：

| 部分 | 你将学会 |
|---|---|
| 一·基础（1–4） | 搭好环境，把金融数据「拿到手、看明白」 |
| 二·金融计量（5–8） | 用统计与计量刻画收益、风险、时间序列、因子与面板 |
| 三·机器学习（9–14） | 监督学习、特征工程、树模型、深度学习、文本 NLP、无监督学习，全程防前视 |
| 四·应用与前沿（15–20） | 高频数据与微结构、组合优化、回测、信用风险，以及大模型与金融智能体 |

!!! tip "怎么读最高效"
    **读一节正文 → 打开该章 notebook 逐格运行 → 改一改参数 → 做章末习题（notebook 末尾有参考解答）**。
    每章顶部都有「在 Colab / Binder 打开」按钮，没装环境也能在云端直接跑。

## 关于代码、数据与可复现

- 所有基础章节依赖**内置示例数据集**，**断网也能跑**；进阶章节提供中国市场真实接口（akshare/tushare）。
- 环境用 [uv](https://docs.astral.sh/uv/) 一键复现，`uv.lock` 锁定全部依赖的精确版本。
- 配套代码、数据、图示、网站与本书同仓库开源，持续更新。
